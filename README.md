Welcome to the Virtual Owl Museum.

Here you can see multiple pictures of owls from the stone age
to the present day.

All images and descriptions are taken from open Internet sources.
The author of the Museum does not claim copyright and does not derive
commercial benefits from the distribution of images and texts.

Also, you can expand the Museum. To do this, copy the picture
in jpg format to the appropriate folder. The first 4 characters
of picture’s name are assigned to the number and are not displayed
when viewing an image in the Museum. The description must have
the same name as the picture. The description must be in utf-8 txt format.
If you want to start a new line, use the code word "NL", to create
an empty line use "NL NL".
